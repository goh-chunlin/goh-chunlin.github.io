About Me

Hello! My name is Chun Lin, a software developer who loves to play with Google APIs and Microsoft technologies.

Currently, I'm participating in Singapore .NET Developers Community as one of the meetup co-organizers. I love to share about my .NET and Microsoft Azure web development journey during the meetups. In November 2016, I deployed first version of our community website which is built using ASP .NET Core on Mac. Check us out at http://dotnet.sg.

Besides being a programmer, I like to blog too. I am a writer on Spiffy from Microsoft Singapore. I also have a personal blog sharing my hobby software development projects.

Work

I am currently a full time software engineer leading a group of nine engineers in Changi Travel Services at Singapore Changi Airport.

/Deployment to Cloud

Introduced Microsoft Azure PaaS platform, i.e. Cloud Service and Web App, which allowed the team to easily host and deploy web applications.

/Changi Airport Transfer

Built an airport transfer system with driver management and SMS notification modules to improve the previous airport transfer workflow in Changi Recommends booths by taking out human intervention.

/SEA System Refactoring

Worked in a small team of three to refactor the operation system supporting Southeast Asia offices by implementing the concept of Factory Design Pattern.

/Serverless Architecture

Experimented and implemented the serverless architecture to transform all schedulers running on virtual machine to Azure Functions which helps to reduce the maintenance cost.

/Internet of Things

Developed the first set of cloud-backed vending machines in Changi Airport. As the system is connected to our cloud servers on Microsoft Azure, admin staff can monitor the stock, transactions, video logs via our online portal easily.

/Team Sharing Sessions

Introduced weekly knowledge sharing sessions where every developer in my team gets a chance to present new knowledge and skills they learnt.

Entertainment Connect
Platform Windows Surface and Windows 8/8.1/10
Language JavaScript

Entertainment Connect allows you to play your songs and videos stored in Microsoft OneDrive directly. It also can keep streamed videos and audios in background and play them later which thus enable the users to switch between different media files in the application smoothly.
