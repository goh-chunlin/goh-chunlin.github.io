About Me

Hello! My name is Chun Lin, a software developer who loves to play with Google APIs and Microsoft technologies.

I'm participating in Singapore .NET Developers Community as one of the meetup co-organizers because I love to share about my .NET and Microsoft Azure web development journey during the meetups.

Besides coding and giving talks, I like to write too. I am a writer on Spiffy from Microsoft Singapore. I also have a personal blog sharing my hobby software development projects.

Career

I am currently a Head of Technology leading a group of engineers in Haulio, incubated by PSA International's corporate venture capital arm, the PSA unboXed.

/Deployment to Cloud

Introduced Microsoft Azure PaaS platform and Visual Studio Team Service, i.e. Cloud Service and Web App, which allows the team to easily host and deploy web applications.

/ASP .NET CORE MVC

Migrated the first version of Haulio core system, the Community, which was written mainly in PHP to a modern ASP .NET Core MVC web application running on Microsoft Azure Web Apps instead of virtual machine.

/Internet of Things

Developed the first set of cloud-backed tracking platform to connect trailer trackers to our cloud server on Microsoft Azure so that hauliers can easily track their trucks on the system.

/Team Sharing Sessions

Introduced daily development sharing sessions where every developer in my team gets a chance to present what they have done and skills they have learned everyday.
